# Additional Requirements for Modern UI

Add these packages to your `requirements.txt` for full functionality:

```txt
# Fake data generation
Faker==18.13.0

# Optional: For production CSS optimization (not required for development)
# django-compressor==4.1.1
```

## Installation

```bash
# Install new packages
pip install Faker==18.13.0

# Or install from requirements
pip install -r requirements-modern.txt
```

## What's Included

### Faker
- Generates realistic fake data for CRM
- Companies, contacts, leads, deals, tasks, emails
- Command: `python manage.py generate_fake_data`

### No Additional Frontend Dependencies!
- Tailwind CSS via CDN
- ApexCharts via CDN
- Vanilla JavaScript (no jQuery, no node_modules needed)

## Optional Enhancements

### For Production CSS Optimization
```bash
pip install django-compressor
```

Add to `settings.py`:
```python
INSTALLED_APPS = [
    # ...
    'compressor',
]

COMPRESS_OFFLINE = True
COMPRESS_CSS_FILTERS = [
    'compressor.filters.css_default.CssAbsoluteFilter',
    'compressor.filters.csscompressor.CSSCompressorFilter',
]
```

### For Real-time Features (Optional)
```bash
pip install django-channels
pip install channels-redis
```

## Docker Support

No changes needed! The project continues to work with Docker:

```bash
docker-compose up
```

All static files are included and available via CDN.

## Environment Setup

No special environment variables needed for the modern UI. The design uses:
- Standard Django static files
- CDN-hosted libraries
- CSS custom properties (variables)
- Browser-native features only
